# 🎓 LinkU - 대학생 스터디 매칭 플랫폼

> **2026학년도 기말고사 대체 과제 (바이브코딩 결과물)**  
> **제출자:** 2026130560 컴퓨터공학과 양준석  
> **개발도구:** Google Antigravity (AI Agent)

## 📌 서비스 개요 및 목적
**LinkU**는 대학생들이 시험 대비, 자격증 취득, 프로젝트 수행 등을 함께 할 스터디원을 쉽고 효율적으로 개설하고 참여할 수 있도록 돕는 **대학생 전용 스터디 매칭 플랫폼**입니다. 

대학 생활 중 스터디 구성원을 구하는 데 소요되는 비효율적인 탐색 비용을 줄이고, 학생들의 전공 분야 및 일정에 특화된 맞춤 스터디를 매칭하는 것을 목표로 합니다.

---

## 🚀 주요 기능 소개

### 1. 회원가입 및 로그인 (인증 및 프로필 생성)
* **학교 이메일 검증**: 대학생 인증을 위해 `@*.ac.kr` 또는 `@*.edu` 형식의 이메일 도메인만 가입할 수 있도록 정규식 형식 검증을 적용했습니다.
* **비밀번호 유효성 및 보안**: 영문, 숫자, 특수문자 조합을 요구하며, 비밀번호 확인 일치 여부를 실시간 검증합니다.
* **학적 정보 입력**: 학과 및 학번을 선택하여 스터디원 간의 소속 신뢰도를 확보합니다.

### 2. 관심 분야 및 선호 일정 설정 (추천 메타데이터 수집)
* **카테고리 및 세부 태그**: 전공, 어학, 자격증, 취업/코딩 등 대분류 설정 및 세부 분야 태그(예: React, TOEIC, SQLD 등)를 커스텀 추가할 수 있습니다.
* **일정 및 장소 선호도**: 선호하는 요일(평일/주말), 시간대(오전/오후), 진행 형태(대면/비대면)를 설정합니다.
* **맞춤형 추천 알고리즘**: 저장된 관심 메타데이터를 기반으로 로그인 시 대시보드에 **'맞춤 추천 스터디'**가 실시간으로 매칭 및 표시됩니다.

### 3. 스터디 개설 및 탐색/필터링
* **스터디 생성**: 방장이 제목, 카테고리, 모집인원, 일정, 상세 규칙/커리큘럼을 입력해 스터디를 생성합니다.
* **다차원 필터링 및 정렬**: 키워드 검색과 함께 진행 방식(온/오프라인), 모집 상태(모집 중/마감) 필터를 조합해 조회할 수 있습니다. 최신 등록순, 인기 조회순, 참여 인원순, 마감 임박순 정렬 UI를 완벽하게 제공합니다.

### 4. 스터디 신청/승인 프로세스 및 실시간 채팅
* **참가 신청 프로세스**: 일반 사용자가 스터디 상세 페이지에서 참여 신청을 보내면 상태가 '대기 중'으로 변경됩니다.
* **알림 센터**: 방장이 로그인하면 대시보드에 **'승인 대기 신청서'**가 표시되어 신청자의 프로필 정보를 확인하고 승인/거절을 결정할 수 있습니다.
* **스터디 전용 채팅방**: 방장의 승인이 완료된 스터디원은 채팅방 입장 권한이 부여되며, 실시간으로 메시지를 전송하고 다른 조원들과의 모의 답변(챗봇 인터랙션)을 통해 작동을 확인할 수 있습니다.

## 🎨 UI/UX 디자인 특징
* **글래스모피즘(Glassmorphism)**: 트렌디하고 세련된 반투명 카드 레이아웃과 딥 인디고 & 퍼플 그라데이션 컬러칩을 적용해 프리미엄 디자인을 구현했습니다.
* **반응형 웹 지원**: 모바일 화면부터 데스크톱 해상도까지 기기 크기에 상관없이 깨지지 않는 플렉스/그리드 레이아웃을 제공합니다.
* **마이크로 인터랙션**: 마우스 호버 시 카드 부드러운 승강 효과, 버튼 스케일링, 알림 배지 및 상태 태그 애니메이션을 추가하여 생동감 넘치는 사용자 인터랙션을 제공합니다.

## 🛠 기술 스택 (Tech Stack)
* **Frontend**: HTML5, CSS3 (순수 CSS 활용 변수 테마 선언)
* **Client Script**: Vanilla JavaScript (ES6)
  * 로컬 실행 시 브라우저 보안에 따른 CORS 에러 방지를 위해 모듈 방식이 아닌 글로벌 네임스페이스 구조로 안정적으로 설계했습니다.
* **Database (Persistence)**: Browser `localStorage`
  * 회원 정보, 스터디 생성 목록, 신청 상태, 읽지 않은 알림, 채팅 대화 데이터 등 모든 사용자 액션이 데이터 유실 없이 브라우저 저장소에 실시간 저장/조회됩니다.

## 💻 로컬 실행 방법

1. 본 저장소의 코드를 다운로드 받거나 `git clone` 합니다.
2. 폴더 내의 `index.html` 파일을 더블 클릭하여 웹 브라우저에서 실행합니다. (서버 구축 불필요)
3. 만약 개발 서버 형태로 실행하고 싶다면 다음 명령어를 실행합니다:
   ```bash
   npm install
   npm run dev
   ```
